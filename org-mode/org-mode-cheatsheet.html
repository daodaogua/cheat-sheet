1 大纲模式

1.1 视图显示

  • S-TAB 全局视图循环
  • TAB 子树视图循环，先由C-u TAB触发，然后TAB使用
  • C-u C-u C-u TAB 显示全部
  • C-x n s 将缓冲区视图局限到当前子树中

1.2 移动

  • C-c C-n 下个标题
  • C-c C-p 上个标题
  • C-c C-f 下个同级标题
  • C-c C-b 上个同级标题
  • C-c C-u 回到上层标题

1.3 结构输入

　- M-RET 在当前级别插入一个项

2 待办事项

  • TODO 当标题以TODO开关时它就成了一个TODO项

2.1 常用命令

  • C-c C-t 切换当前项状态
  • C-c / t 将buffer折叠，像是TODO和它们所在的层次的标题

2.2 复选框

　- [] 放在任务步骤之前可以很合适的将任务分步

3 时间戳

时间戳有激活<2019-01-08 二>和非激活时间戳[2019-01-08 二]

3.1 创建时间戳

  • C-c . 插入或修改时间戳
  • C-c ! 插入或修改未激活的时间戳

　- S-LEFT/RIGHT 将光标处理的时间戳改变一天

　- <>–<> 时间段

3.2 日程安排

  • C-c C-s SCHEDULED: <2019-01-08 二>
  • C-c C-d DEADLINE: <2019-01-08 二>
